Simulation Data

The SCREAMv0 simulation data is available on NERSC HPSS.

If you have an account on NERSC, you can retrieve the data locally or remotely using Globus.

To download simulation data locally on a NERSC machine, refer to NERSC’s documentation here.

If you do not have access to NERSC, you can download simulation data directly through the NERSC HPSS web interface. Note that this will be slow given the large file sizes.

Scripts originally used to run SCREAMv0 DYAMOND2 simulations are available here.
