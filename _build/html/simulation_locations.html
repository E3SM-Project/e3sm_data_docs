Simulation Locations

To download simulation data run:

zstash extract --hpss=globus://9cd89cfd-6d04-11e5-ba46-22000b92c6ec/<HPSS path below>

For more information, refer to zstash usage.

If you do not have Globus access, you can download a simulation directly here.

Simulations have NERSC HPSS paths at:

/home/projects/e3sm/www/WaterCycle/E3SMv2/LR

The preferred retrieval method is ESGF. All the native output is available at ESGF.

Original run scripts (the scripts that were originally used to create the simulations) are here.
